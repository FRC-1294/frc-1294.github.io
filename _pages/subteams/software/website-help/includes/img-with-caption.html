How to Use img-with-caption

img-with-caption is a Jekyll include. It is a file in the project, located at _includes/img-with-caption.html.

This include is for creating a figure, an image with a <p> for a caption, and links the image to it's src url for viewing the full image. All of this structure is done in one {% raw %}{% include %}{% endraw %}.

This code when placed into a page will create the below image and caption, as seen on the electrical page.
{% highlight liquid %} {% raw %} {% include img-with-caption.html file="/assets/img/subteams/electrical/frc_3128_wiringdiagram.png" alt="FRC Control System Wiring Diagram" caption='A new, up to date wiring diagram for the FRC control system by team 3128 -- Found on Chief Delphi' %} {% endraw %} {% endhighlight %}
{% include img-with-caption.html file="/assets/img/subteams/electrical/frc_3128_wiringdiagram.png" alt="FRC Control System Wiring Diagram" caption='A new, up to date wiring diagram for the FRC control system by team 3128 -- Found on Chief Delphi' %}
img-with-caption takes three parameters: file, alt, and caption. file and caption are pretty self explanitory: the path to the image and the caption for the image respectively. alt is "an alternative text description of the image", typically for use in the event the image can't be loaded, accessiblity, or some other reason a text representation of an image is needed.

In the event one of these parameters is ommited, defaults are used, but they generally aren't wanted, as for most pages they look pretty silly. {% highlight liquid %} {% raw %} {% include img-with-caption.html %} {% endraw %} {% endhighlight %}
